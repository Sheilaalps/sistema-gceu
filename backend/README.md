#  Sistema de Gestão GCEU

Este projeto é uma ferramenta completa para gestão de membros, cargos e usuários de grupos familiares (GCEU). O objetivo é demonstrar o domínio de diferentes tecnologias de Back-end e Front-end.

##  Estrutura do Projeto

O repositório está organizado em subpastas para facilitar o desenvolvimento:

* **/backend-java**: API Rest construída com Java 17 e Spring Boot (JPA, MySQL/H2).
* **/backend-node**: (Em desenvolvimento) API construída com Node.js e Express.
* **/frontend-react**: (Em desenvolvimento) Interface web moderna com React + Vite.

---

##  Tecnologias Utilizadas

### Back-end (Java)
- Java 17
- Spring Boot 3
- Spring Data JPA
- Maven
- Banco de Dados: MySQL / H2 (Testes)

### Ferramentas de Teste
- **Insomnia / Postman**: Para testes de endpoints.
- **Git/GitHub**: Controle de versão.
