📦 BACKEND - MELHORIAS

✅ Rotas Completadas (10 endpoints)

  POST   /users/login       - Login com JWT
  POST   /users/registrar   - Criar usuário (admin)
  GET    /users/perfil      - Perfil autenticado

Membros:
  GET    /membros           - Listar com paginação
  GET    /membros/:id       - Buscar específico
  POST   /membros           - Criar novo
  PUT    /membros/:id       - Atualizar
  DELETE /membros/:id       - Deletar
  POST   /membros/:id/presenca  - Registrar presença
  GET    /membros/status/:status - Filtrar por status

  - auth.js: Geração e verificação de JWT

✅ Banco de Dados
  - Schema Prisma com 2 modelos
  - Enums para status e níveis
  - Script seed.js com dados iniciais
  - Credenciais padrão criadas

✅ Dependências Adicionadas
  - bcryptjs: Criptografia
  - jsonwebtoken: JWT
